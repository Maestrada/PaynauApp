# PaynauApp API

## Requerimientos

- .NET 8.0
- Docker
- MySQL

## Instrucciones de Ejecución

1. Clona el repositorio.
2. Navega al directorio del proyecto.
3. Ejecuta `docker-compose up` para iniciar la API y la base de datos.
4. Accede a `http://localhost:5000/swagger` para ver la documentación de Swagger.
